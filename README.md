### 一、产品概述
**“****瑞金特医****”是一款由专业医疗机构支持、以****医学营养干预为核心****的智能健康管理平台，专注于为慢性病患者（如糖尿病、高血压、肥胖症）、术后康复人群及亚健康个体提供****科学、个性化、可持续的营养与体重管理服务****。平台融合临床医学、营养学与人工智能技术，打造集“在线评估—定制方案—行为干预—数据追踪—效果反馈”于一体的全流程健康管理闭环。**

**该应用依托权威医疗资源背景，结合AI营养师、智能体脂秤、饮食记录、运动指导等功能，帮助用户实现科学减重、血糖控制、代谢改善等健康目标。平台不仅面向个人用户，也支持医疗机构、社区卫生中心、健康管理机构开展远程慢病管理项目，推动“****以治疗为中心****”向“****以预防与管理为中心****”的医疗模式转型。**

**“瑞金特医”不仅是健康管理工具，更是医学营养干预的数字化实践载体，致力于让每一个人都能享受到专业级的营养照护。**

### 二、核心功能
1. **营养师****  
****提供一对一专业营养师咨询服务，根据用户体质、疾病状况、生活习惯制定个性化饮食与生活方式建议，支持图文、语音沟通。**
2. **智能机器人****  
****内置AI营养助手，可自动回答常见问题（如“吃这个会升糖吗？”“如何控制热量？”），提供即时、便捷的营养指导。**
3. **优选商城****  
****推荐符合医学营养标准的食品、代餐、营养补充剂等商品，支持在线购买，确保食材安全可控。**
4. **消息提醒****  
****定时推送饮食建议、服药提醒、运动计划、体检预约等通知，帮助用户养成良好习惯。**
5. **饮食日记****  
****支持手动或拍照录入每日三餐，系统自动分析总热量、碳水、蛋白质、脂肪摄入情况，生成营养报告。**
6. **食物查询****  
****内置海量食物数据库，输入名称即可查看其营养成分（热量、糖分、GI值等），便于控制摄入。**
7. **社交圈****  
****用户可加入健康社群，分享减重经验、饮食成果、运动打卡，形成正向激励与同伴支持。**
8. **资讯****  
****发布权威科普文章，涵盖糖尿病饮食、减肥误区、肠道健康、代谢综合征等内容，提升用户健康素养。**
9. **智能体重秤****  
****支持连接蓝牙智能体脂秤，自动同步体重、体脂率、肌肉量、水分、基础代谢等多维度指标。**
10. **体重秤指标****  
****展示各项身体成分数据，并提供健康参考范围与趋势分析图，帮助用户科学解读体检结果。**
11. **个人主页****  
****查看个人信息、健康档案、历史记录、成就勋章等，构建专属健康画像。**
12. **记体重****  
****手动记录每日体重变化，支持图表展示，直观反映减重进展。**
13. **记运动****  
****记录运动类型、时长、消耗卡路里等信息，支持步数同步、跑步/骑行轨迹导入，全面追踪身体活动。**
14. **记腰围****  
****支持测量并记录腰围数据，用于评估腹部脂肪堆积情况，预防代谢综合征。**
15. **我的目标****  
****设置个性化健康目标（如“3个月减重5kg”“空腹血糖控制在6.5以下”），系统将根据目标生成执行计划并跟踪完成度。**
16. **首页推荐****  
****（主界面功能）展示用户当前状态（如体重、BMI）、今日任务、营养建议、热门课程等，突出“医学营养减重”主题。**
17. **AI营养评估****  
****（隐含功能）通过问卷采集用户基本信息后，AI自动生成营养风险评估报告，识别潜在健康隐患。**
18. **远程随访支持****  
****（可扩展功能）支持医生或营养师定期查看用户数据，进行远程随访与调整治疗方案，适用于慢病管理项目。**

### 三、产品价值
#### **用户价值：让健康管理有依据、有方法、可持续**
+ **科学减重不反弹****：基于医学营养原理设计饮食方案，避免极端节食，实现健康、持久的体重管理。**
+ **精准控糖更安心****：为糖尿病患者提供低GI饮食建议、餐后血糖预测、用药提醒等服务，有效控制病情。**
+ **专业指导更放心****：由注册营养师或临床医生参与制定方案，确保安全性与有效性，避免盲目跟风。**
+ **数据驱动更高效****：通过智能设备与App联动，实时监测身体变化，动态调整策略，提升干预效果。**
+ **家庭共享更贴心：支持为家人创建健康档案，父母可为子女或老人管理饮食与运动，实现全家健康守护。**

## 架构特点
### 架构特性
+ **前后端分离架构，独立开发，符合主流开发模式**
+ **前端以Vue3+Vite为主技术，AntdV为UI界面框架**
+ **后端SpringBoot3为基础，MybatisPlus为数据操作框架，Redis为缓存框架**
+ **Maven多模块管理，插件化开发，方便安装、卸载、升级，降低耦合**
+ **业务模块与API抽离，模块之间便捷引用**
+ **数据库设计精巧，字段规范、易于扩展**
+ **支持国产密码算法加解密，等保测评国产项目无压力**
+ **支持MYSQL、ORACLE、SQLSERVER、PGSQL等主流标准结构式数据库**
+ **支持达梦、人大金仓、南大通用、九有、瀚高、虚谷数据库等国产数据库**
+ **支持中创、宝蓝德、东方通等中间件**
+ **支持Windows、Linux操作系统、国产操作系统部署**

### 功能特性
+ **完善的系统基础功能，满足使用需求，避免重复造轮子**
+ **支持本地文件、阿里云文件、腾讯云文件、MINIO文件上传**
+ **支持本地邮件、阿里云邮件、腾讯云邮件发送**
+ **支持阿里云短信、腾讯云短信发送**
+ **B、C端双账号认证体系，会话治理各自独立**
+ **完善的登录日志、操作日志、异常日志等审计功能**
+ **完善的会话监控、数据源监控、系统监控等必备监控功能**
+ **支持组织机构、权限管理、定时任务、系统配置等基础功能**

### 安全特性
+ **采用SaToken轻量级 Java 权限认证框架，功能强大、学习成本低**
+ **支持登录认证、权限认证、单点登录、三方登录、OAuth2.0等认证模式**
+ **增强的RBAC权限设计，资源于接口独立授权，更加灵活**
+ **支持按钮级别细粒度独立授权，界面按钮动态展示**
+ **支持API接口注解式、路由拦截式鉴权，防止越界访问**
+ **独创的数据范围机制，每个接口都可以配置不同数据范围**
+ **支持限流防抖，防重复提交，有效阻止脏数据产生**
+ **密码、手机、身份证号等使用国密算法加密传输、加密存储，数据更安全**
+ **操作日志使用SM2进行完整性保护，满足安全审计要求**

### 界面特性
+ **Vue3 + Vite为基础，AntdV为界面UI框架，视觉风格清新简洁**
+ **精细化设计，注重界面的每一处细节，操作轻松友好**
+ **暗黑风格、经典菜单、双排菜单、多页签、目录坞、主题切换等功能应有尽有**
+ **统一的网络框架、API接口拦截框架，拿来即可上手**
